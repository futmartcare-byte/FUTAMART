# FutaMart

A buying and selling marketplace app with real-time chat between buyers and sellers.

## Features
- Browse and search listings
- Create and edit listings
- Real-time chat with sellers/buyers
- Notifications
- User profiles and saved listings

## Getting Started
1. Install dependencies: `npm install`
2. Run the app: `npm run dev`

## Tech Stack
- React (Vite)
- Tailwind CSS
- Supabase (database, auth, real-time)
